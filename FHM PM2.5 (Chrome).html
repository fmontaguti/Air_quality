Anlsie de Dados

Com a base PM2.5 onde foram observados 420768 registros dentro de 5 anos a respeito de poluentes e particulados no ar dentro de algumas estaes da China, alm de informaes como temperatura, direo do vento, presso, precipitao de chuva, etc. Foram realizadas algumas anlises para entender o comportamento dos dados.

Primeiramente tomou-se como fator a estao de origem para verificar como que os poluentes estavam alocados geograficamente. A medida de referncia foi a mediana para a concentrao dos poluentes, uma vez que no possui tanta interferncia de medidas extremas.

Uma anlise de correlalo tambm foi realizada entre os dados registrados, para verificar a influncia de uma varivel sobre a outra. Seu resultado foi apresentado em uma matrix de correlao envolvendo 11 das 17 variveis do Dataset.

Com o enfoque em PM2.5, nota-se uma correlao forte com as variveis PM10, SO2, NO2, CO e tambm com a velocidade do vento [WSPM].

Para se realizar um modelo preditivo de concentrao de PM2.5 com base em outras variveis, utilizou-se as de maior correlao encontrada. Envolvendo 5 variveis de maior correlao, os dados do modelo podem ser observados abaixo.
